### 1. 캠페인 삭제 버튼 기능 미구현 ⚠️

**위치:** `src/app/dashboard/campaigns/page.tsx:240-247`

**문제:**
```tsx

  variant="ghost"
  size="sm"
  aria-label={`${campaign.title} 삭제`}

  <span className="sr-only">삭제</span>
</Button>
// ❌ onClick 핸들러 없음 - 버튼이 작동하지 않음
```

**개선 필요:**
- 삭제 핸들러 추가
- 확인 다이얼로그 추가
- 에러 처리 추가

### 3. 작은 버튼들의 터치 타겟 ⚠️

**문제:**
```tsx
<Button size="sm" />  // 32px 높이 - 44x44px 미만
```

**발견 위치:**
- 캠페인 페이지 액션 버튼들
- 여러 테이블의 액션 버튼

**개선 필요:**
- 모바일에서만 size="default" 사용
- 또는 min-h-[44px] 강제 적용
