## ♿ 접근성 테스트

### 키보드 네비게이션
1. Tab 키로 모든 버튼 이동 가능
2. Enter/Space로 버튼 활성화
3. Escape로 모달/드롭다운 닫기
4. 화살표 키로 검색 자동완성 이동

### 스크린 리더 (선택)
- macOS: VoiceOver (Cmd+F5)
- Windows: NVDA (무료)
