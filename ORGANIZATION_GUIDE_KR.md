## 🚀 시작하기

### 회원가입 및 로그인

1. **회원가입**
   - 메인 페이지에서 "기관 가입" 버튼을 클릭
   - 기본 정보 입력: 이메일, 비밀번호
   - 이메일 인증 완료
   - 기관 프로필 완성

2. **로그인**
   - 이메일과 비밀번호로 로그인
   - 로그인 후 자동으로 대시보드로 이동

### 대시보드 개요

로그인 후 기관 대시보드에서 다음 기능들을 확인할 수 있습니다:

- **📊 캠페인 관리**: 생성한 캠페인 목록 및 현황
- **📋 프로포절 관리**: 받은 제안서 검토 및 관리
- **💬 메시지**: 전문가와의 실시간 소통
- **🔔 알림**: 중요한 업데이트 알림
- **👤 프로필 관리**: 기관 정보 수정

## 👥 기관 프로필 설정

### 필수 정보 입력

완성도 높은 프로필은 더 나은 매칭 결과를 가져옵니다:

#### 1. 기본 정보 (필수)
```
✅ 조직명 (정확한 법인명 또는 상호명)
✅ 대표자명
✅ 업종/사업분야
✅ 직원 수
```

#### 2. 추가 정보 (선택)
```
□ 사업자등록번호
□ 담당자 직책
□ 웹사이트 URL
□ 조직 소개
```

### 프로필 완성하기

**경로**: 대시보드 → 조직 프로필

**완성도 진행률 표시**:
- 프로필 작성 시 실시간 진행률 표시 (8개 항목 기준)
- 필수 항목 4개 + 선택 항목 4개
- 프로필 완성 후 `is_profile_complete` 상태 업데이트

**지원 업종**:
- IT/소프트웨어
- 제조업
- 금융/보험
- 의료/헬스케어
- 교육
- 유통/리테일
- 건설/부동산
- 미디어/엔터테인먼트
- 컨설팅
- 기타

**직원 수 범위**:
- 1-10명
- 11-50명
- 51-100명
- 101-300명
- 301-1000명
- 1000명 이상

## 📝 캠페인 생성 및 관리

### 캠페인 생성 단계별 가이드

**경로**: 대시보드 → 캠페인 → "새 캠페인 만들기"

#### 1단계: 기본 정보 설정

**캠페인 제목 작성 요령:**
```
✅ 구체적이고 명확한 제목
✅ 주요 기술이나 분야 포함
❌ "전문가 구합니다" (모호함)
✅ "React 기반 전자상거래 플랫폼 개발 전문가"
```

**상세 설명 작성 가이드:**
```markdown
## 프로젝트 개요
- 목적 및 목표
- 배경 정보
- 기대 성과

## 업무 내용
- 구체적인 작업 범위
- 주요 업무
- 결과물 형태

## 요구 사항
- 필수 기술 스킬
- 경력 조건
- 우대 사항

## 프로젝트 일정
- 시작일
- 마일스톤
- 완료 예정일
```

#### 2단계: 캠페인 유형 선택

**지원하는 캠페인 유형:**
- **멘토링/강의**: 지속적인 조언 및 가이드
- **투자 매칭**: 투자자 및 파트너 연결
- **서비스 아웃소싱**: 프로젝트 기반 서비스

#### 3단계: 카테고리 및 키워드 설정

**키워드 선택 팁:**
- 구체적인 기술명 사용
- 동의어 및 관련 용어 포함
- 배열 형태로 여러 키워드 입력 가능

#### 4단계: 예산 및 일정 설정

**예산 설정:**
- 최소 예산 (budget_min)
- 최대 예산 (budget_max)
- 예산 미정 시 null 가능

**일정 설정:**
- 시작일 (start_date)
- 종료일 (end_date)
- 필요한 전문가 수 (required_experts)

#### 5단계: 추가 정보

**지역 설정:**
- 근무 지역 지정 (선택사항)
- 원격 가능 여부

**캠페인 상태:**
- `draft`: 임시저장
- `active`: 진행중 (게시됨)
- `completed`: 완료
- `cancelled`: 취소됨

### 캠페인 관리 기능

**캠페인 목록에서 가능한 작업:**
- 📊 상세보기: 캠페인 전체 정보 확인
- ✏️ 수정: 캠페인 정보 업데이트
- 🗑️ 삭제: 캠페인 삭제
- 📈 통계: 조회수, 지원 수 등 확인

**필터 및 검색:**
- 상태별 필터: 모든 상태, 진행중, 임시저장, 완료, 취소됨
- 검색: 제목, 설명, 키워드로 검색

## 📋 프로포절 관리

### 프로포절 검토 가이드

**경로**: 대시보드 → 프로포절

#### 프로포절 정보 확인

**제안서에 포함된 정보:**
```
✅ 제안 내용 (proposal_text)
✅ 예상 예산 (estimated_budget)
✅ 예상 시작일 (estimated_start_date)
✅ 예상 종료일 (estimated_end_date)
✅ 포트폴리오 링크 (portfolio_links)
✅ 전문가 정보
```

#### 프로포절 상태 관리

**프로포절 상태:**
- **pending** (검토중): 아직 검토되지 않은 제안서
- **accepted** (승인됨): 승인된 제안서
- **rejected** (거절됨): 거절된 제안서
- **withdrawn** (철회됨): 전문가가 철회한 제안서

**프로포절 액션:**
1. **승인**: 제안서 승인하기
2. **거절**: 제안서 거절하기 (거절 사유 입력 가능)
3. **메시지 보내기**: 전문가와 직접 소통
4. **상세보기**: 전체 제안서 내용 확인

### 프로포절 통계

**대시보드에서 확인 가능한 통계:**
- 📊 전체 제안서 수
- ⏳ 검토 대기 중
- ✅ 승인된 제안서
- ❌ 거절된 제안서
- 📈 응답률
- ⏱️ 평균 응답 시간

## 💬 메시징 및 소통

### 메시지 시스템

**경로**: 대시보드 → 메시지

#### 메시지 스레드 관리

**메시지 스레드 정보:**
- 대화 상대 정보 (이름, 조직명, 이메일)
- 관련 캠페인
- 마지막 메시지 내용
- 읽지 않은 메시지 수
- 마지막 메시지 시간

**필터 옵션:**
- 모든 대화
- 읽지 않음
- 캠페인별
- 제안서별

#### 효과적인 커뮤니케이션

**초기 연락 시 포함할 내용:**
```markdown
안녕하세요, [전문가명]님

[캠페인명] 프로젝트에 관심을 가져주셔서 감사합니다.

**프로젝트 개요**
- 목적: [구체적인 목적]
- 기간: [예상 기간]
- 예산: [예산 범위]

**질문사항**
1. [구체적인 질문1]
2. [구체적인 질문2]

**다음 단계**
프로포절 검토 후 [언제]까지 연락드리겠습니다.

감사합니다.
[담당자명], [기관명]
```

**정기 소통 가이드라인:**
- 주간 프로젝트: 매일 간단한 업데이트
- 월간 프로젝트: 주 2-3회 진행 상황 확인
- 장기 프로젝트: 주 1회 정기 미팅

## ❓ 문제해결 및 FAQ

### 자주 묻는 질문

#### Q1. 캠페인을 올렸는데 지원이 적어요.
**A1. 다음 사항들을 점검해보세요:**
- 제목이 구체적이고 매력적한가?
- 예산이 시장 가격과 비교해 적정한가?
- 요구사항이 너무 까다롭지 않은가?
- 키워드가 전문가들이 검색할만한 용어인가?

#### Q2. 전문가와 연락이 안돼요.
**A2. 단계별 대응 방법:**
1. 플랫폼 메시지 시스템 이용
2. 24시간 후에도 응답이 없으면 두 번째 메시지
3. 72시간 후에도 무응답이면 다른 전문가 고려

#### Q3. 프로필 완성도는 어떻게 높이나요?
**A3. 프로필 완성 팁:**
1. 모든 필수 항목 입력 (조직명, 대표자명, 업종, 직원 수)
2. 선택 항목도 최대한 입력 (사업자번호, 웹사이트, 조직 소개 등)
3. 조직 소개는 상세하게 작성 (200자 이상 권장)
4. 정기적인 정보 업데이트

#### Q4. 캠페인 상태는 언제 변경하나요?
**A4. 상태 변경 가이드:**
- **draft → active**: 캠페인 준비 완료 후 게시
- **active → completed**: 프로젝트 완료 시
- **active → cancelled**: 프로젝트 취소 시

### 기술 지원

#### 플랫폼 사용 문제:
- **로그인 문제**: 비밀번호 재설정 기능 이용
- **프로필 저장 오류**: 모든 필수 항목 입력 확인
- **캠페인 생성 실패**: 프로필 완성 여부 확인
- **알림 안옴**: 이메일 스팸 폴더 확인
